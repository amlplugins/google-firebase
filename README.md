# @amlplugins/google-firebase

Native `firebase` SDK re-exported under the `@amlplugins` namespace.

> Firebase Web SDK (client).

## Install

```bash
npm install @amlplugins/google-firebase
```

## Use

```ts
import { /* named exports from firebase */ } from "@amlplugins/google-firebase";
// or
import sdk from "@amlplugins/google-firebase";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
